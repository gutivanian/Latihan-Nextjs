Next JS

Next JS adalah framework berdasar React dengan kapabilitas render dari sisi server. Ini sangat cepat dan SEO friendly. Menggunakan Next JS kita bisa membuat aplikasi React lebih mudah dan cepat.

Next JS memiliki banyak kelebihan:

  1. Hot Code Reload - Dapat mendeteksi perubahan pada file dan melakukan reload secara otomatis
  2. Automatic Routing - Tidak perlu melakukan konfigurasi URL untuk routing. File hanya perlu diletakkan dalam halaman folder. Semua url akan dipetakan ke file system. Kustomisasi juga dapat dilakukan
  3. Component Spesific Styles - jsx dengan style memberikan style ke komponen global atau spesifik
  4. Server Side Rendering - Komponen react di render lebih awal di server sehingga memuat lebih cepat di client
  5. Node Ecosystem - Next js adalah berdasar react, sehingga berkaitan erat dengan ekosistem node
  6. Automatic Code Split - Next js merender halaman dengan library yang dibutuhkannya. Next js ketimbang membuat sebuah file javascript besar, lebih memilih untuk membuat banyak sumber daya. Ketika halaman dirender, hanya javascript yang dibutuhkan yang dimuat.
  7. Prefetch - prefetch memungkinkan Next.js untuk mengunduh dan mempersiapkan sumber daya yang dibutuhkan untuk halaman tujuan lebih awal, sehingga ketika pengguna akhirnya mengklik tautan, halaman tersebut dapat dimuat lebih cepat.
  8. Dynamic Components - Next js mengijinkan kita untuk mengimport javascript module dan komponen react secara dinamis. Ini berarti Anda dapat menunda pemuatan suatu komponen atau modul sampai komponen atau modul tersebut benar-benar dibutuhkan. Fitur ini sangat berguna untuk mengoptimalkan kinerja aplikasi dengan mengurangi ukuran bundle awal dan mempercepat waktu muat halaman.
  9. Export Static Page - Next.js memungkinkan Anda untuk mengekspor aplikasi web Anda sebagai situs statis penuh. Ini berarti Anda dapat mengonversi aplikasi Next.js menjadi kumpulan file HTML, CSS, dan JavaScript statis yang dapat di-host di server mana pun atau layanan hosting statis seperti Vercel, Netlify, atau GitHub Pages.
  10. Typescript Support - Next js mendukung typescript

Instalasi

Karena Next js adalah framework React, kita bisa menggunakan lingkungan Node. Kita bisa menginstall React dan Next bersamaan. 

        npm install next react react-dom
maka di dalam package.json akan berisi 
        {
            "name": "nextjs",
            "version": "1.0.0",
            "description": "",
            "main": "index.js",
            "dependencies": {
               "next": "^9.4.4",
               "react": "^16.13.1",
               "react-dom": "^16.13.1"
            },
            "devDependencies": {},
            "scripts": {
               "test": "echo \"Error: no test specified\" && exit 1",
               "dev": "next",
               "build": "next build",
               "start": "next start"
            },
            "author": "",
            "license": "ISC"
         }

Membuat File Halaman

Kita bisa membuat sebuah folder dan membuat index.js. 

        function HomePage() {
           return <div>Welcome to Next.js!</div>
        }
        
        export default HomePage

Menjalankan Server Next.js

Untuk menjalankan server, kita bisa gunakan 

        npm run dev
